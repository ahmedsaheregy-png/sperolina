# صفحة هبوط DXN - سبيرولينا وجانوديرما

صفحة هبوط احترافية باللغة العربية لمنتجات DXN (الاسبيرولينا والجانوديرما).

## المحتويات

- **index.html** - الصفحة الرئيسية
- **style.css** - ملف التنسيقات
- **script.js** - ملف الجافاسكريبت للتفاعلات
- **الصور:**
  - `hero-bg.png` - صورة الخلفية الرئيسية
  - `products.png` - صورة منتجات الاسبيرولينا
  - `ganoderma.png` - صورة منتجات الجانوديرما
  - `thumb1.png`, `thumb2.png`, `thumb3.png` - صور مصغرة مخصصة لفيديوهات الاسبيرولينا
  - `gano_thumb1.png`, `gano_thumb2.png`, `gano_thumb3.png` - صور مصغرة مخصصة لفيديوهات الجانوديرما

## المميزات

✅ تصميم عصري وجذاب باللغة العربية (RTL)
✅ متجاوب مع جميع الأجهزة (Responsive)
✅ صور مخصصة احترافية لجميع الفيديوهات
✅ أزرار واتساب للتواصل المباشر
✅ رسوم متحركة سلسة
✅ قسم فيديوهات تعليمية للمنتجين
✅ تحسين محركات البحث (SEO)
